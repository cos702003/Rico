This project is a slideshow viewer app that retrieves images from a web url and displays them fullscreen one at a time to the Display monitor.
Please read the Eboard_Setup_Directions for proper installation and operation.
This application uses the Raspbian Stretch image along with the PIL, REQUESTS, and BYTESIO libraries as dependencies.
This application also requires the intallation pf PQIV, a linux-based slide show viewer.
